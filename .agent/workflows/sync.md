// turbo-all
1. Check git status to see modified files
   `git status`

2. Stage all changes
   `git add .`

3. Commit changes with a generic or generated message (Assistant will generate)
   `git commit -m "Auto-sync: Applied recent updates and fixes"`

4. Push to main branch
   `git push origin main`
